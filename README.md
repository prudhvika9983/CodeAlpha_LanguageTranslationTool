# CodeAlpha Language Translation Tool

This project is a Python-based Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

It allows users to enter text, select source and target languages, and get the translated output using a translation API.

## Features
- User-friendly interface
- Supports multiple languages
- Uses translation API
- Displays translated text clearly
- Organized project structure

## Technologies Used
- Python
- Flask
- Translation API
- HTML, CSS

## How to Run
1. Install required packages:
   pip install -r requirements.txt

2. Run the application:
   python translator.py

3. Open browser and go to:
   http://127.0.0.1:5000/
